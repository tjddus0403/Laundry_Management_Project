# Laundry_Management_Project
2020 가을학기 프로그래밍및실습2 프로젝트입니다.

## 프로그램 기능
- 파일 불러오기
- 파일 저장하기
- 종료하기
- 옷 추가
- 옷 정보 출력
- 옷 정보 수정
- 옷장 열기
- 세탁 방법 찾기

## 프로그램 설치 및 실행 방법
```bash
git clone https://github.com/tjddus0403/Laundry_Management_Project
cd Laundry_Management_Project
make
./cloth_manage
```
